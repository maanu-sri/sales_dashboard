# 📊 Sales Analytics Dashboard

An interactive sales analytics dashboard built with Python and Streamlit.

## Features
- Revenue trend analysis (monthly)
- Sales breakdown by category and region
- Top 10 products by revenue
- Dynamic filters — region, category, date range
- KPI cards — total revenue, orders, units sold

## Tech Stack
- Python
- Streamlit
- Plotly
- Pandas

## Run Locally
```bash
pip install streamlit pandas plotly
python -m streamlit run app.py
```
